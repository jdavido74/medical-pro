## Task 5: Backend — Planning Service Room Conflict Checks

- [ ] **Step 1: Add checkRoomAvailability helper function**

After the existing `getExistingAppointments` function, add:

```javascript
/**
 * Check if a room is available at a given time
 * @param {Sequelize} clinicDb
 * @param {string} roomId - Room UUID
 * @param {string} date - YYYY-MM-DD
 * @param {string} startTime - HH:MM
 * @param {string} endTime - HH:MM
 * @param {boolean} exclusiveRequired - true if treatment needs exclusive room access
 * @param {string|null} excludeId - appointment ID to exclude (edit mode)
 * @returns {boolean} true if room is available
 */
async function checkRoomAvailability(clinicDb, roomId, date, startTime, endTime, exclusiveRequired, excludeId = null) {
  if (!roomId) return true; // No room assigned = no constraint

  // Check if room is active
  const Room = await getModel(clinicDb, 'Room');
  const room = await Room.findByPk(roomId);
  if (!room || !room.is_active) return true; // Inactive room = no constraint

  let sql;
  const replacements = { roomId, date, startTime, endTime };

  if (exclusiveRequired) {
    // Exclusive treatment: room must be completely free
    sql = `SELECT COUNT(*) as cnt FROM appointments a
           WHERE a.room_id = :roomId AND a.appointment_date = :date
           AND a.status NOT IN ('cancelled')
           AND (a.start_time < :endTime AND a.end_time > :startTime)`;
  } else {
    // Non-exclusive: only blocked by exclusive treatments
    sql = `SELECT COUNT(*) as cnt FROM appointments a
           JOIN products_services ps ON a.service_id = ps.id
           WHERE a.room_id = :roomId AND a.appointment_date = :date
           AND a.status NOT IN ('cancelled') AND ps.exclusive_room = true
           AND (a.start_time < :endTime AND a.end_time > :startTime)`;
  }

  if (excludeId) {
    sql += ' AND a.id != :excludeId';
    replacements.excludeId = excludeId;
  }

  // Sequelize query() returns [rows, metadata] — destructure correctly
  const [rows] = await clinicDb.query(sql, { replacements });
  const result = rows[0];
  return parseInt(result.cnt) === 0;
}
```

- [ ] **Step 3: Integrate room check into getMultiTreatmentSlots**

In `getMultiTreatmentSlots`, add `alreadyClaimedRooms` Map alongside `alreadyUsedInSlot`. For each segment:
1. Get `roomId` from assigned machine
2. Check room against DB (existing appointments)
3. Check room against `alreadyClaimedRooms` (prior segments in same chain)
4. Add segment to `alreadyClaimedRooms`

```javascript
const alreadyClaimedRooms = new Map(); // roomId -> [{ startTime, endTime, exclusive }]

// Inside the segment loop, after machine is assigned:
const segRoomId = assignedMachine.room_id || null;
if (segRoomId) {
  // Check against DB
  const roomOk = await checkRoomAvailability(clinicDb, segRoomId, date, segStartTime, segEndTime, treatment.exclusive_room);
  if (!roomOk) { segmentFailed = true; break; }

  // Check against already-claimed rooms in this chain
  const claimed = alreadyClaimedRooms.get(segRoomId) || [];
  for (const prev of claimed) {
    if (timeRangesOverlap(segStartTime, segEndTime, prev.startTime, prev.endTime)) {
      if (treatment.exclusive_room || prev.exclusive) {
        segmentFailed = true; break;
      }
    }
  }
  if (segmentFailed) break;

  // Claim this room segment
  if (!alreadyClaimedRooms.has(segRoomId)) alreadyClaimedRooms.set(segRoomId, []);
  alreadyClaimedRooms.get(segRoomId).push({ startTime: segStartTime, endTime: segEndTime, exclusive: treatment.exclusive_room });
}
```

**IMPORTANT:** Declare `let alreadyClaimedRooms = new Map();` INSIDE the outer `for (let currentStart = ...)` loop, BEFORE the inner `for (const treatment of treatmentData)` loop. This ensures it resets for each candidate start time. Do NOT declare it outside the outer loop or it will accumulate state across different candidate times and silently block valid slots.
